All Git-Hub cheat sheet below here...
1.Username create in git-bash. 
git config --global user.name "yourname"


2.Username display in git-bash. 
git config user.name


3.Useremail create in git-bash. 
git config --global user.email "youremail"


4.Useremail display in git-bash. 
git config user.email


5.List information in git-bash. 
git config --list


6.Display git folder status. 
git status


7.Folder convert in git repository. 
git init


8.All files add in git repository. 
git add --a


9.For save process. 
git commit -m "initial commit"


10.All commits display. 
git log


11.Only one file add in repository. 
git add filename.txt


12.Delete the repository from git bash. 
rm -rf .git


13.Online repository pull process. 
git clone {paste url}


14.Display present working directory. 
pwd


15.Display command list. 
ls


16.Apply for current directory. 
cd directorypath/


17.Create new file in git bash. 
touch pa.log


18.Create gitignore file. 
touch .gitingnore


19.Difference between staged file and working directory. 
git diff


20.Difference between last commit. 
git diff --staged


21.Only track file commit. 
git commit -a -m "track file commited"


22.For remove file. 
git rm filename.txt


23.Rename file name. 
git mv first.txt second.txt


24.Track file convert to untrack file. 
git rm --cached filename.txt


25.All commit information display. 
git log -p


26.Only last 2 commit display. 
git log -p -2


27.Display commit in short information. 
git log --start


28.Different format display commit. 
git log --pretty=oneline
git log --pretty=short
git log --since=2.days
git log --since=2.week
git log --pretty=format:"%h--%ae"
git log --pretty=format:"%an"


29.Commit information changed. 
git commit --amend

git restore --staged filename.txt

32.After last commit file checkout. 
git checkout -f

git config --global alias.st status

34.Short form for commit.

36.Create branch. 
git checkout -b branchname


37.Move to master branch. 
git checkout master


38.Merge two branch. 
git merge branchname


39.Dispaly all branch. 
git branch


40.Display branch last commit. 
git branch -v


41.Display merged branch. 
git branch --merged


42.Display no-merged branch. 
git branch --no-merged


43.Delete merged branch. 
git branch -d branchname


44.Delete no-merged branch. 
git branch -D branchname


45.For push branch. 
git push origin branchname


46.Duplicate branch on git-hub website. 
git push origin branchname1:branchname2


47.Delete branch on git-hub website. 
git push -d origin branchname


48.For pull branch.
